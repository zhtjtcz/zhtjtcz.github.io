解析

相邻的到障碍的距离相减，得到障碍之间的距离。根据两个人的数据可以得到两个数列，判断两个数列是否相同即可。

注意数列的顺序可以变换，要对所有的顺序分别判断。

代码

C

解析

预处理出每一行的光标到达三种类型的密码所需移动的距离。然后暴力穷举，枚举所有可能的排列情况，时间复杂度为$O(C_{N}^{3}*3!)$。

代码

D

解析

已知C序列的大小顺序和对应A序列的值，那么按照大小顺序排序。因为B中每个数都有一个范围，所以可以假设(排序后)第一个B元素的取值范围为[l,r]，然后线性向后扫描，根据C序列的大小关系，在扫描过程中B序列元素的取值范围一定在不断缩小，如果在某个位置为空集，那么无解。

扫描完毕后让最后一个B元素取最小值，倒着还原B序列。此时的B序列不一定为答案，还需要进行调整。显然B序列同时增加或减少一个常数，仍然可以满足题意。所以如果还原出的B序列中的最小值小于l，那么对整个数列进行调整，直到满足题意为止。此时的B序列即为答案。
